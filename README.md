# Automation Script Pack

Contains three small console scripts:
- rename_files.py
- extract_data.py
- send_email.py

## Requirements
Python 3, and for send_email.py the network access and valid SMTP credentials.

## Run any script like:
python rename_files.py
python extract_data.py
python send_email.py

## Notes
- For send_email.py use app passwords (Gmail) or SMTP credentials. Keep them secret.
- extract_data.py reads a text file and prints found emails & phone numbers.
